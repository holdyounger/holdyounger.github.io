基础知识点

A 和 AAAA

A 和 AAAA 都是DNS记录，它们的作用是将域名解析到IP地址。具体来说，A记录将域名解析到IPv4地址，而AAAA记录将域名解析到IPv6地址。

因此，主要的区别在于 A 记录用于IPv4，而 AAAA 记录用于IPv6。
